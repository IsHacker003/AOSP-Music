# AOSP Music


Fork of [AOSP stock music app](https://github.com/aosp-mirror/platform_packages_apps_music) which was included in Android 2.3, ported to work in Android Studio and newer Android versions and with added features such as:

 - Lockscreen / notification music controls
 - Slightly refreshed "Now playing" screen.
